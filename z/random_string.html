0 默认 数字+大小写字母+特殊字符
1 不包含特殊字符
2 只有数字
3 不包含容易混淆的字符 ( l o O )
4 不包含逃逸字符 ( ' " )

l 长度，默认16

t 类型，默认0

e 排除字符，默认为空

url中必须转义的字符
字符    URL编码
空格    %20
"      %22
#      %23
%      %25
&      %26
'      %27
(      %28
)      %29
+      %2B
,      %2C
/      %2F
:      %3A
;      %3B
<      %3C
=      %3D
>      %3E
?      %3F
@      %40
\      %5C
|      %7C

其他字符可以根据ascii表选择转义
